# Ejercicio 08: gestión de productos

## CLASES
PRODUCTO name
PRECIO valor
TRANSPORTE tipoTransporte
PEDIDO diaentrega(dia/mes/año)

## Relaciones
Relacionar PRODUCTO->PRECIO
Relacionar PEDIDO->PRODUCTO
Relacionar PEDIDO->TRANSPORTE 

## Acciones
Cambiar el nombre de PRODUCTO;
Cambiar el PEDIDO;

## Listas
Cambiaremos que el PEDIDO no solo tenga un PRODUCTO si no una lista de ellos

## Añadir elementos a la lista
Permitir añadir nuevos productos a la lista de PEDIDO
